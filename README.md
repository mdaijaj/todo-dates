# todo-dates

## Requeirement:-
1. backend: Nodejs with express framework <br> 
2. fronted: Reactjs with Bootstrap <br>
3. database: Monogodb with Mongoose

`Backend:-` <br> 
git clone repo <br> 
install all dependency <br> 
start project <br> 


`Fronted:-` <br> 
Change fronted folder <br> 
install all dependency <br> 
start project
